<H3> L-IBMS : Loei Investment Budget Management System : ระบบบริหารจัดการงบลงทุนและงบค่าเสื่อม สสจ.เลย </H3>

📊 1. แดชบอร์ดภาพรวม (Main Dashboard)</br>
หน้าต่างบานแรกสำหรับผู้บริหาร เพื่อดูภาพรวมของงบลงทุนทั้งหมด</br>
สรุปตัวเลขสำคัญ (KPI Cards): แสดงจำนวนรายการและวงเงินรวม แยกประเภท "ครุภัณฑ์" และ "สิ่งก่อสร้าง"</br>
Interactive Map: แผนที่จังหวัดเลยแบบ Choropleth แสดงความหนาแน่นของงบประมาณตามอำเภอ (สีเข้ม = งบเยอะ) พร้อม Tooltip เมื่อเอาเมาส์ชี้</br>
กราฟแท่งเปรียบเทียบ: สัดส่วนงบลงทุนแบ่งตามประเภทหน่วยบริการ (รพ., บริหาร, ปฐมภูมิ)</br>
ตารางสรุปข้อมูลเชิงลึก: แยกข้อมูลตามกลุ่ม (ครุภัณฑ์/สิ่งก่อสร้าง) จัดกลุ่มตามประเภทหน่วยบริการ และเรียงลำดับอำเภอแบบ Custom (เมืองเลยขึ้นก่อน ตามด้วยอำเภออื่นๆ ตามที่กำหนด)</br>
</br>
🏗️ 2. ระบบติดตามและอัปเดต งบลงทุน (Investment Budget)</br>
หน้าสรุปความก้าวหน้า (Progress Dashboard):</br>
แยกส่วนสิ่งก่อสร้างและครุภัณฑ์ชัดเจน</br>
แสดงร้อยละความสำเร็จของโครงการ (เปอร์เซ็นต์รายการที่เบิกจ่ายเสร็จสิ้น)</br>
ระบบแจ้งเตือนความเสี่ยง (Alerts): คัดกรองรายการที่ล่าช้า เสี่ยงพับ หรือรอเบิกจ่าย มาแสดงให้เห็นเด่นชัด</br>
กราฟโดนัทแสดงสัดส่วนสถานะงาน 8 ขั้นตอน</br>
ตารางแสดงแถบความก้าวหน้า (Progress Bar) และคำนวณวันสิ้นสุดสัญญาอัตโนมัติ (ขึ้นป้ายเตือนสีแดง/เหลืองถ้าใกล้หมดเวลา)</br>
หน้ารายงานความก้าวหน้า (Report):</br>
ดูตารางข้อมูลทั้งหมด และมีปุ่ม "แก้ไข" เป็น Modal เด้งขึ้นมาให้กรอกข้อมูลอัปเดตสถานะ, ยอดเงิน, และวันที่ต่างๆ รายโครงการ</br>

🏥 3. สรุปผลการดำเนินการ งบค่าเสื่อม (UC Dashboard) ✨ [อัปเกรดล่าสุด]</br>
Smart KPI Cards: สรุปวงเงินรวม, เบิกจ่ายแล้ว, คงเหลือ และแสดง Badge %เบิกจ่าย (แดง, ส้ม, เขียวอ่อน, เขียวเข้ม ตามเกณฑ์เป้าหมาย)</br>
Advanced Charts (กราฟขั้นสูง):</br>
กราฟโดนัท: แสดงสัดส่วนเงินตามสถานะ 0-5 โดยย่อตัวเลขให้อ่านง่ายเป็นหน่วย "ล้านบาท (ลบ.)"</br>
กราฟ Dual-Axis (แท่งผสมจุด): แกนซ้ายแสดงกราฟแท่งจำนวนรายการ (แยกสีตามสถานะ) ซ้อนกับแกนขวาแสดงจุด (Dot) ร้อยละการเบิกจ่าย</br>
ตารางสรุปแม่ข่าย (Smart Data Table):</br>
จัดกลุ่มตาม รพ.แม่ข่าย ค่าเริ่มต้น เรียงตามรหัส 5 หลัก (10705, 11030... ไปจนถึง รพ.สต.)</br>
ไฮไลต์ตัวเลขร้อยละด้วยสีดำเข้ม และใช้ Badge สีสะดุดตากับช่อง %เบิกจ่าย</br>
คลิกหัวตารางเพื่อ "เรียงข้อมูล (Sorting)" สลับไปมาได้ (ตามชื่อ หรือ ตาม %เบิกจ่าย)</br>
ตารางแยกสังกัด (สป.สธ. และ อบจ.):</br>
คิดความก้าวหน้าแบบ "Milestone" (เอาสถานะ 0-5 มาคำนวณเป็นคะแนนเนื้องาน) ทำให้เห็นความคืบหน้าแม้จะยังไม่มีการเบิกจ่ายเงิน</br>

📝 4. รายงานและระบบอัปเดต งบค่าเสื่อม (UC Report) ✨ [ไฮไลต์เด็ด]</br>
On-Demand Rendering: โหลดหน้าเว็บเร็วติดจรวด เพราะตารางจะไม่ทำงานจนกว่าผู้ใช้จะกดปุ่ม "🔍 ค้นหาข้อมูล"</br>
UI/UX ตารางข้อมูล:</br>
ปลดล็อกความสูงตาราง (ปล่อยยาวลงมาไม่ต้องมี Scroll ซ้อน)</br>
แสดง Badge สังกัด (สป.สธ. สีเขียว / อบจ. สีฟ้า) ท้ายชื่อแม่ข่าย</br>
ย่อสถานะเป็น "Badge ตัวเลขวงกลม (0-5)" พร้อมแถบ Legend อธิบายสีด้านบน</br>
หัวตารางคลิกเพื่อจัดเรียง (Sort) ได้ทุกคอลัมน์สำคัญ</br>
📝 ระบบอัปเดตข้อมูลแบบเหมาเข่ง (Batch Update): ติ๊กเลือกหลายรายการเพื่อเปลี่ยนสถานะพร้อมกันผ่าน Modal พร้อมระบบเช็กยอดเงินคงเหลือไม่ให้ติดลบ</br>
📥 ระบบนำเข้าด้วย Excel (Excel Data Import):</br>
อัปโหลดไฟล์ Excel เข้าไปเช็กกับระบบด้วยรหัสรายการ (ID)</br>
Smart Data Diff: ระบบจะฉลาดพอที่จะคัดกรองเฉพาะรายการที่ "สถานะ" หรือ "เงินเบิกจ่าย" เปลี่ยนแปลงมาโชว์ (รายการไหนเหมือนเดิมจะถูกข้ามเพื่อลดภาระ)</br>
มี หน้าต่าง Preview เทียบข้อมูล "ซ้าย (ของเดิม)" vs "ขวา (ของใหม่)" พร้อมไฮไลต์สีเขียวจุดที่เปลี่ยน ให้ผู้ใช้ติ๊กเลือกและกดยืนยันก่อนบันทึกลง Database รวดเดียวจบ</br>

🔍 5. ระบบค้นหาประวัติคำขอ & Service Plan</br>
ค้นหาประวัติคำขอ (Search): สำหรับค้นหาข้อมูลย้อนหลัง กรองด้วยเงื่อนไขต่างๆ แบ่งหน้าแสดงผล (Pagination) และมี Badge สีบอกผลการพิจารณา (อนุมัติ/ไม่อนุมัติ)</br>
Service Plan: ตารางสรุปข้อมูลแผนบริการล่วงหน้า (ปี 2571-2573) สรุปวงเงินรวม และความต้องการแยกตามสาขา SP</br>

สรุปจุดเด่นทางเทคนิค (Tech Highlights):
No-Reload Experience: สลับหน้าและโหลดข้อมูลผ่าน Single Page Application (SPA) ไม่ต้องรีเฟรชหน้าเว็บ
Data Diff & Batch Write: อ่าน Excel จากหน้าเว็บด้วย SheetJS และเขียนทับข้อมูลปริมาณมากลง Google Sheets ในเสี้ยววินาที
Responsive Design: รองรับการใช้งานทั้งบนหน้าจอคอมพิวเตอร์และแท็บเล็ต (Tailwind CSS)
